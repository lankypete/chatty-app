Chatty
======

A minimal multi user messenger application.

### Stack
React
Webpack
WebSockets
SASS

### Install
1. Pull project
2. Pull server project (here)[https://github.com/lankypete/chatty-app-server]
3. npm install
4. run on localhost:3000
